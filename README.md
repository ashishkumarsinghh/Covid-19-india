# Covid-19-india

Covid-19-India stats and summary.

# Tech Stack

- React
- create-react-app
- Bootstrap & React-Bootstrap
- ChartJS

# How to Run ?

- git clone git@github.com:ashishkumarsinghh/Covid-19-india.git
- cd Covid-19-india/covid-19-india
- yarn
- yarn start
- Open Browser (localhost:3000)
